Задание 2

Вам даны этапы проекта. Необходимо поставить их в верном порядке.

Важно!

Чтобы решить эту задачу необходимо ознакомиться с теорией жизненного цикла проекта

Определение стратегии

описание требований, оценки их реализации, бюджета и возможности выполнения работ для клиентов на взаимных условиях.

Анализ

исследование определенных функций, связей и атрибутов.

Проектирование

сбор данных.

Реализация

разработка продукта, взаимодействие команды для достижения оставленных целей.

Тестирование

испытание продукта.

Внедрение

запуск продукта, тестирование клиентами.

Техническая поддержка

постоянное взаимодействие с клиентами.
